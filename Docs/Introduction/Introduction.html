The Fidget Microcontroller

An Introduction

It’s a micro-what?  A “microcontroller”?  What the heck is that?

A microcontroller is actually a small computer that is primarily used to “sense” conditions in the physical world (inputs) and then “control” external devices (outputs).  Inputs can be as simple as a button or as complex as a GPS receiver and outputs can be just as varied… from blinking LEDs to generating control signals for brushless motors.  Unlike a powerful  desktop computer though, this computer is usually just a single chip, with limited speed and memory and is usually dedicated to a single purpose.  It’s the kind of computer they use in microwave ovens to do things like read the keypad and control the oven’s cooking.

What can you do with one?

The uses of such a device are quite numerous because it is programmable and supports a number ways to interact with the environment.  But in this case, the Fidget is more like a versatile electronic toy.  It can blink lights, makes sounds, sense light and temperature among other things.  It can use push buttons and other inputs to allow a person to interact with the device.  I have implemented a method of separating the parts such that it can be used for a variety of fun activities.  To do something different with the Fidget, you simply load another program!  Here are some examples of what is possible…

  • Mini Multi-Color Light Show
  • Security Alarm
  • Electronic Music Box
  • Fake Keyboard
  • etc...

Ok, how do I get started?

First you will need a Mini USB cable.  These are an older style cable that should be readily available.  Make sure that it is an actual “data” cable and not just a “power” cable.

The only PC setup involved is to install the drivers for the Fidget.  Once this is done, you can use the rest of the software on this disk to start programming the Fidget.

  1. From this disk run /WindowsDriver/InstallDriver.exe to install the driver

Once it has been installed you are ready to go..

So how do I do that?

  1. Start the programming software by running /Arduino/arduino.exe from this disk.  It should appear rather quickly...


  2. Select a program from File -> Examples -> Fidget_Examples -> ???
    Note:It will open it in another instance of the Arduino window
  3. In the new instance,
  1. Select Tools -> Board -> Digispark (Tiny Core)
  2. Select Tools -> Programmer -> Digispark
  1. Prepare your Fidget for programming by attaching the Comm Board to the Base Board

  2. Now select File -> Upload.
    It will take a few moments, but eventually the message at the bottom of the window should say…

         Running Digispark Uploader…
        Plug in device now… (Will timeout in 60 seconds)

  3. Plug in the Fidget.
    It will take a few moments, but eventually the message at the bottom of the window should say...

         >Starting the user app …
        running: 100% complete
        >> Micronuclues done. Thank you!

  4. Now unplug the Fidget.
  5. Remove the Comm Board and replace it with the board appropriate for the program that you selected.
  6. Plug in the power cable with the silver side of the plug facing out.
  7. The program will begin running after a few seconds.  

Enjoy!
